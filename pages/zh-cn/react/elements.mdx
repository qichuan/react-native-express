## `createElement`

<Details>一般来说我们会使用 JSX 语法来创建 **React 元素**</Details>
，但你也可以直接调用 `React.createElement` API 来实例化一个 React 元素。

这个方法的签名为：`createElement(type, props, children)`。

## 元素内部

从上面示例的控制台里可以看到，一个 React 元素是一个至少包含 `type` 和 `props` 两个属性的 JavaScript 对象。

- `type` 是一个渲染 UI 元素的函数
- `props` 是一个可以包含任意属性的对象，里面有例子里的 `foo` 属性，和包含嵌套的 React 元素的 `children` 属性。<Details> `children` 属性有些特殊，因为它是从代码中的嵌套 JSX 元素自动创建的。</Details>

`ref` 和 `key` 并不属于 props，我们会在后续章节里详细解释。
