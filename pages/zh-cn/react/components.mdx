## 什么是组件？

React **组件** 是用来创建 UI 的可复用构建块，也是 React 元素的`类型`。

除了使用内置组件，如：`View`, `Text`, 和 `Button`。我们也可以创建自定义组件。

> 组件和元素的关系有点像 OOP 里的类和实例。如果把 `View` 组件当成类来看的话，那么调用 `React.createElement(View)` 就是创建一个实例（元素）。

---

## 定义组件

我们可以通过两种方式定义 React 组件：

- **函数组件** - 以参数（`props`）为输入并返回 React 元素的函数
- **类组件** - 继承 `React.Component` 并实现 `render` 函数的类

> 接下来我们分别看看定义 React 组件的这两种方式。
>
> 函数组件 API 是推荐的 React 组件定义方式，也是我们介绍的重点。在一些较旧的文档教程里，你还是看到类组件 API 的使用示例。
