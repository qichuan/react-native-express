React 里大部分的输入组件都是无状态的。输入组件的属性通常包括一个 `value`，和一个以 `onChange` 为前缀的回调函数。我们可以使用这两个属性来操控输入组件，且无需直接访问底层的 UI 实现。

---

## 输入栏

我们来看看一个文字输入栏的案例。

`text` 是一个**状态变量**，用来存储输入栏的当前值。我们把 `text` 作为 `value` 传入到 `TextInput` 中，每当 `onChangeText` 被调用时，使用 `setText` 来更新状态变量。

> 我们使用 [useState](/react/hooks/usestate) hook 来存储文字值，在后续章节里我们会详细介绍 hook。
