## React 工作原理

React 让我们可以使用 JavaScript 对象树的形式来定义应用程序的整个 UI。这些 JavaScript 对象也叫做 **React 元素**，<Details>并由 React 提供的一系列工具来创建和管理</Details>


React Native 调用系统原生 UI 组件在手机屏幕上**渲染** React 元素树。<Details>并且，React 元素树和系统原生 UI 组件有绑定关系，每当某个元素属性被更改后，对应的原生 UI 组件也会自动更新。</Details>
